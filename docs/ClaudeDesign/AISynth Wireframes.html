SynthVibe / AI Synth VST
v0.1 · lo-fi · 7 variations
1200 × 750 plugin window

Seven different takes on the plugin window. Each one solves the same puzzle — natural-language sound design, 8-voice engine, AI preset history — but reaches for a different product personality and parameter metaphor.

Interactions stay consistent across all seven: the AI prompt lives in a pull-down modal that returns 3–4 variations to A/B; every prompt becomes an auto-saved preset; a mini keyboard sits at the bottom; a live oscilloscope is always running somewhere.

Try the Tweaks button (bottom-right) to swap accent colors and toggle the sketchy annotations.

Design system

  • Type: Inter (UI) · JetBrains Mono (values, labels) · Caveat (handwritten annotations)
  • Palette: warm paper base (#fbf9f4), ink (#1c1b1a), one accent per variant
  • Primitives: boxes, dashed placeholders, stroked knobs, monospace param readouts
  • Reference: Vital-clean information density, but rougher edges + our own voice

Next steps once we pick a direction

  • Lock one or two favorite layouts, then do a higher-fidelity pass with real type hierarchy and final spacing.
  • Design the AI pull-down modal flow (empty state → generating → 3–4 variation cards → apply).
  • Design the preset history drawer (auto-saved every prompt, with the prompt text visible).
  • Define the section-accent color system across OSC / Filter / Env / LFO / FX / Arp.
